# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. All participants — maintainers, contributors, and community members — are expected to follow it in issues, pull requests, discussions, and any other project spaces.

You can read the full text here: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

In short: be respectful, assume good faith, focus on what's best for the community, and show empathy toward other community members.

## Reporting

If you experience or witness behavior that violates the Code of Conduct, please report it privately by sending a GitHub direct message to the maintainer [@j0elmiller](https://github.com/j0elmiller). Reports will be handled confidentially.

For security vulnerabilities, see [SECURITY.md](SECURITY.md) instead — they use a different reporting channel.

## Enforcement

The maintainer is responsible for clarifying and enforcing these standards. Consequences for violations follow the enforcement ladder described in the Contributor Covenant (correction, warning, temporary ban, permanent ban), applied proportionately to the behavior.
